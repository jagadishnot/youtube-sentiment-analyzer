# 🎥 YouTube Sentiment Analyzer

AI-powered web application that analyzes YouTube comments to understand audience sentiment and engagement.

## 🚀 Features
- 📊 Sentiment Analysis (Positive / Negative / Neutral)
- 🤖 AI Insights Summary
- 🎥 Embedded Video Player
- 📈 Interactive Charts (Recharts)
- ⚠ Smart Ad Detection
- 💬 Comment Analysis

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js + Express
- AI: HuggingFace Transformers (RoBERTa)
- Charts: Recharts

## 💡 Key Highlights
- Real-time YouTube data extraction
- Transformer-based NLP sentiment model
- AI-generated audience feedback summary
- Full-stack integration
